Weather sensor support

This part of the documentation describes the weather screen and what you can (and sometimes cannot) do with weather sensors. Since version 1.8 LamPI does support weather station sensors. At first only the UPM WT440H (outdoor) sensor is supported and it works fine. Other snesors can be added to the LamPI-receiver/sniffer program later, but it is good to know that the LamPI-daemon offers support for receiving Json messages from weather sensors.

Json message

The Json message that is sent to the LamPI-daemon is as follows:

message = {
tcnt : integer,
type : 'json',
action : 'weather',
brand: wt440h # or other supported brand
address: integer, # (house code)
channel: integer,
temperature: integer.integer
humidity: intteger,
windspeed: integer,
winddirection: integer, # in degrees
rain : integer # (mm per hour)

};

Adding a new Weather sensor

Weather sensors can be added to the LamPI-receiver by making a sensor function in C and including it in the receiver.c file and in the Makefile.
