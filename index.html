LamPI

LamPI is a Home Automation project for switching equipment in the house over 444 MHz senders/transmitters on a RaspberryPI.

As you see, the GUI interface is designed to operate (and manage) all tasks for the connected devices and minimize the need for manual changing of configuration files etc. let alone making configuration changes at compile time.

LamPI Introduction

The goal of my project was to provide an easy interface to the family to ontrol the lights in or house. We had klikaanklikuit.nl switches and we have an iPad (and some Android phones and Laptops) and want to control these switches with an easy-to-use GUI interface. This mean I needed at least 2 components: A component the would control transmitter and receiver and handle low-level device commands, and a GUI in Javascipt/jQuery that would handle the human interaction. In order to store configuration information and for timing commands and unattended use, we need a daemon process that works in the background as well as a database (written in MySQL) to store configuration information.
