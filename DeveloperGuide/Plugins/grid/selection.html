Grid system

As an alternative for teh standard LamPI layout we looked at a grid system that could server the GUI elements of LamPI. It would need to show the sitwhces, dimmers, thermostats and sensors in a tile which would then enable the user to filter and sort this environment.

Ideally, when double clicking the tile, the user then would be able to change the valu of the underlying element.

What to choose

We see several examples of grid based systems at the moment in for example Metrod (window 8), Google+, Office on-line, Pinterst etc, Although these tile-grid systems are becoming more and more popular in commercial systems, it is hard to choose one in a non-commercial hobby environment. As always, we hope that the plug-in or library is of high quality, it-s use if free or unlimited for our purpose and it is maitaned over a logger time. It would also be nice if we get some sort of support from the developer should anything go wrong.

The ones to choose from (at the moment) are:

  • Masonry
  • isotope (also commercial)
  • Shapeshift
  • gridster
  • gridstack

Masonry

Isotope

Advantages

Shapeshift

Shapeshift is one of the smaller companies in the row. Shapeshift is made by "we the media". Unfortunately it seems like they are inactive now for a while (since 2013) and the webside referred to on gridhub does not work anymore.

 

Gridster

Advantages of Gridster

  • Completely free to use
  • Resizable
  • Draggable

Most important drawbacks of Gridster are:

  • No automatic resizing of the grid if the screen is resized
  • Performance problems if we create large amount of widgets
  • Documentation is not simple and lacks examples
  • Support of 'Ducksboard' has stopped since they are bought by another company

Gridstack

By Pavel Reznikov

According to the git site, Gridstack is inspired by Gridster. Interface looks more or less the same and probably gridster code can be ported to gridstack without too much of a hasle.

Advantages:

  • Looks like gridster successor
  • Issues seem to be addressed and there is acive support
  • Dynamic
  • Resizeable
  • Draggable

Disadvantages

  • Not so many peope following (yet)
