1. Infrared Remote

I bought an Infrared Remote board for the Raspberry.

Features:

1.infrared receiver function

  Operating frequency: 38K HZ 
  Receiving distance: 18-20m 
  Reception angle: +/- 45 degrees


2.infrared transmit function

   Wavelength: 940nm
   Transmission Distance: 7-8m


3.supports infrared dual LED transmit 

4.support LIRC software, using LIRC and expansion board, the user can be used to copy all of the infrared remote control function (TV, amplifier, DVD, etc. Electric remote control), and to control your various electrical devices by command .

5.Support XBMC system, users can use the infrared function expansion board in the XBMC environment

6.supports dual GPIO button, the user can configure the button functions by programming

7.The pin out is as follows:

Signal RPi_connector(P4)
Button1 -> Pin13
Button2 -> Pin15
IR_IN -> Pin11
IR_OUT -> Pin12
