SI4432 Transceiver

I bought this transceiver some time ago on eBay. It supports FSK and OOK transmission, and having transmitter and receiver together in one package I expected some advantages compared to the traditional 433MHz transmitters and receivers that in use until now. Of course extended range being the first one, but also making use of the "RadionHead" libraries it is possibel to create mesh networks and reliable communication. Using better 433MHz protocols (FSK) it is possible to send larger data packages in a reliable fashion.

    Package

    The SI-4432 transceiver is the same as the RFM22 transceiver from HopeRF sold by Sparkfun and others.

    Pin definitions:


    1. CND power supply
    2. GPIO0 internal module, to control tranmitter antenna
    3. GPI01 internal access module to control receiver antenna
    4. GPIO2 directly connected to the chip GPIO2 pin
    5. VCC positive supply 3.3V
    6. SDO SPI data out
    7. SDI serial data input. 0-VDD V digital input, this pin for 4-wire serial data clock function
    8. SCLK SPI clock in
    9. NSEL Chip select in
    10. NIRQ interrupt output pin
    11. SDN Shutdown pin, should be connected to GND or the module goes to deep sleep.
    12. GND access power ground
    13. ANT 50 ohm coaxial antenna
    14. GND
